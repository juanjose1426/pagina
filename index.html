¿Qué es CSS y por qué es importante en el desarrollo web?

CSS (Hojas de Estilo en Cascada) es un lenguaje que se usa para darle estilo y diseño a las páginas web. Mientras que HTML se encarga de organizar y estructurar el contenido de una página (como texto, imágenes y enlaces), CSS se ocupa de cómo se verá ese contenido, controlando cosas como el color, el tamaño, la ubicación de los elementos y las animaciones. En resumen, HTML define el "qué" de la página, y CSS define el "cómo" debe lucir.

Importancia de CSS en el desarrollo web

CSeparación de contenido y presentación: CSS permite separar la estructura de la página (HTML) de su diseño (estilos). Esto facilita la gestión, mantenimiento y actualización del sitio web, ya que solo se necesita modificar los estilos sin tocar el contenido HTML.

La Importancia de JavaScript en la Web

JavaScript es un lenguaje de programación crucial en el desarrollo web moderno, y su importancia radica en que hace que las páginas web sean interactivas, dinámicas y capaces de ofrecer una experiencia de usuario más rica y atractiva

  • Interatividad:Menús, animaciones
  • Aplicaciones modernas: Usado en frameworks como React, Vue y Angular
  • Carga de datos en tiempo real: Busquedas instantáneas, actualizaciones sin recargar la pagina
  • Compatibilidad univrsal: Funciona en todos los navegadores
  • Juegos y animaciones webun uso en Three.js, WebGL y GSAP

    • JavaScript es la base de la web moderna y sin el, no existiria Muchas de las aplicaciones que usamos hoy como las redes sociales.

    Tabla de Atributos de CSS

    Esta tabla contiene información sobre distintos atributos CSS y su funcion

    Nombre del atributo Descripción Valores posibles Ejemplo de uso Imagen representativa Fuente de información
    font-size Define el tamaño de la fuente del texto. px, em, rem, %, vw font-size: 16px; MDN
    border-radius Redondea las esquinas de un elemento. px, %, em border-radius: 10px; MDN
    display Define el tipo de caja de visualización del elemento. block, inline, flex, grid, none display: flex; MDN
    flexbox (display: flex;) Habilita un contenedor flexible para alinear elementos. row, column, wrap, justify-content, align-items display: flex; justify-content: center; MDN
    position Define la forma en que se posiciona un elemento. static, relative, absolute, fixed, sticky position: absolute; MDN
    box-shadow Añade sombras a los elementos. x-offset, y-offset, blur, spread, color box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5); MDN
    overflow Controla el desbordamiento del contenido de un elemento. visible, hidden, scroll, auto overflow: scroll; MDN
    z-index Define la profundidad de un elemento en la pantalla. Número entero (positivos y negativos) z-index: 10; MDN
    opacity Define la transparencia del elemento. Valores entre 0 (transparente) y 1 (opaco) opacity: 0.5; MDN
    margin y padding Define el espacio exterior (margin) e interior (padding) del elemento. px, %, em, auto margin: 10px; padding: 5px; MDN
    writing-mode Define la dirección del texto en un elemento. horizontal-tb, vertical-rl, vertical-lr writing-mode: vertical-rl; MDN
    resize Permite redimensionar un elemento por el usuario. none, both, horizontal, vertical resize: both; MDN
    clip-path Define una forma de recorte para un elemento. circle(), ellipse(), inset(), polygon() clip-path: circle(50%); MDN
    pointer-events Controla si un elemento puede recibir eventos del mouse. auto, none pointer-events: none; MDN
